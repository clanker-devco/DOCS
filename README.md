# Clanker Documentation

Welcome to the official Clanker documentation. Clanker is a token deployment protocol on Base, Arbitrum, and other EVM chains.

## Quick Links

- **[API Reference](api-reference/README.md)** — REST API for token search, deployment, market data, and more.
- **[SDK Reference](sdk-reference/README.md)** — TypeScript SDK for on-chain token deployment and management.

## Getting Started

### API

The Clanker API is available at `https://www.clanker.world/api`. Public endpoints require no authentication. Partner endpoints require an API key passed via the `x-api-key` header.

### SDK

Install the SDK:

```bash
npm install clanker-sdk
```

See the [SDK Reference](sdk-reference/README.md) for full usage documentation.
